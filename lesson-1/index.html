Lesson 1

CRUD Operations in DB Browser

Lesson Overview

In this lesson, you will learn how to use DB Browser to perform CRUD operations (Create, Read, Update, Delete) on a sample database.

Topics Covered
  • What is a database
  • Opening DB Browser
  • Creating a new database
  • Creating tables and columns
  • Inserting data into tables
  • Viewing and editing data
  • Deleting data
What You'll Need
  • DB Browser for SQLite
  • Sample database file
What is a database and where does DB Browser come in?

A database is a structured collection of data. It is used to store, manage, and information. DB Browser is a free, open-source tool that allows you to create, edit, and manage databases. It is user-friendly and provides a graphical interface to interact with databases.
